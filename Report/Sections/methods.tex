\section{Methods}
Purpose: Describe how the experiment was performed so others could reproduce it.\\
Content:
\begin{itemize}
    \item Experimental apparatus (schematics, photos, diagrams)
    \item Sample preparation or materials used
    \item Measurement techniques (instrumentation, detection methods)
    \item Calibration procedures
    \item Any special conditions (temperature, vacuum, magnetic field, etc.)
\end{itemize}
Tips: Be clear, concise, and logical. Use sub-sections if your setup is complex. Include uncertainty sources.
